# Nom du Projet : Météo

# la Description

Une application web permettant d'afficher la météo en temps réel tel que l'etat actuelle , les prévision pour toute la semaine avec le max et le min de température et les prévisions pours les dix prochaines heures.

# Technologies utilisées

HTML5 / CSS3 (Flexbox & Grid)
JavaScript (ES6)
API Météo (Open-Meteo)

# Fonctionnalités principales

Recherche par ville
Affichage des icônes selon le code météo (WMO)
Design responsive (Mobile/Desktop)

# Lien du rendu final

Cliquez ici pour voir le projet en ligne: https://eyamaghraoui.github.io/maghraoui_eya_M-t-o/

# Nouveautés explorées

Appris à manipuler les codes WMO pour changer les icônes dynamiquement.
Utilisation de Fetch API pour récupérer des données JSON.

# Difficultés rencontrées

Problème d'affichage des icônes quand le code ne correspondait à rien.
Alignement des éléments du forecast en CSS Grid.

# Solutions apportées

Ajout d'une condition par défaut (fallback) dans la fonction JavaScript.
Utilisation de media queries pour ajuster la grille sur mobile.
